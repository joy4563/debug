1. What is the difference between map, forEach , filter and find?

The main difference between map() and forEach() is the returning value. The forEach() method returns undefined and map() returns a new array.
filter() returns an array all the element that satisfies the condition, but find() return the first element that satisfies the condition.

3. What is Event loop and how it is work?

The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks. If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.
